         <td class="wovenodd-col1">
            <strong>Remarks</strong>
         </td>
         <td class="wovenodd-col2">
            <p>Since a reading can be a multi-measure section, the 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/scoreDef">scoreDef</a>
               element is allowed so that a reading may have its own meta-data without incurring
               the
               overhead of child 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/section">section</a> elements. The 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/app">app</a>
               sub-element is permitted in order to allow nested sub-variants.This element is modelled
               on an element in the Text Encoding Initiative (TEI)
               standard.
            </p>

         <td class="wovenodd-col1">
            <span class="label" lang="en">Remarks</span>
         </td>
         <td class="wovenodd-col2">
            <p>The child elements of this element are treated as parts of the bibliographic entity
               containing the 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/componentGrp">componentGrp</a>. Although this is an implicit way of
               expressing FRBR's hasPart / isPartOf-relationships, it avoids this terminology in
               order to
               prevent confusion with musical terminology. All children of a component must be the
               same
               type as its parent: only 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/work">work</a> children are allowed within 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/work">work</a>, 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/item">item</a> in 
               <a class="link_odd_elementSpec" href="/{{ page.version }}/item">item</a>,
               etc.
            </p>
         </td>
      </tr>
